### Week 1 Objectives:

* Get acquainted with the AWS cloud platform and understand core service categories (Compute, Storage, Networking, Database).
* Set up IAM, AWS CLI, and build a foundational VPC with public/private subnets.

### Week 1 Achievements:

* Understood AWS global infrastructure: Regions, Availability Zones, and Edge Locations.
* Created an AWS Free Tier account and configured IAM with least-privilege policies.
* Installed and configured AWS CLI with Access Key, Secret Key, and default Region.
* Mastered VPC fundamentals: CIDR notation, public/private subnets, route tables, and Internet Gateway.
* Deployed a custom VPC and verified connectivity from a public EC2 instance.
* Understood the difference between Security Groups (stateful) and NACLs (stateless).
